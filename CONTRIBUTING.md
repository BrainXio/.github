# Contributing to BrainXio Projects

Thanks for showing up with curiosity — that's already half the work.

We build with questions, honesty, and a bit of humor. If you're here to dig deeper, fix something, add an idea, or just explore, you're welcome.

## Quick Start

1. Fork & clone the repo you're interested in.
2. Create a branch: `git checkout -b feat/my-cool-idea` or `fix/bug-i-found`
3. Make changes — small and focused is best.
4. Commit with Conventional Commits (e.g. `feat: add new prompt template`, `fix: config loading edge-case`).
5. Push and open a PR.

## What We Love

- Questions that challenge assumptions
- Clear, honest explanations
- Small, testable PRs
- Humor in comments/docs (if it fits)
- Empathy — we all learn at different speeds

## What to Expect

- We'll review with kindness and curiosity
- We may ask questions to understand your intent
- We squash-merge for clean history
- All contributions matter — even ideas or typos

No contribution is too small. Question everything, support everyone, and let's build something real — together.

Got questions? Open an issue or chat in discussions.
